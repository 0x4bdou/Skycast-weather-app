# SKYCAST

A modern weather dashboard that displays real-time weather conditions, air quality data, and a 5-day forecast for any city.

The application provides a clean dark UI and supports multiple languages.

## Features

* Real-time weather information
* Temperature, humidity, wind speed, and visibility
* Air quality index and pollution indicators
* Sunrise and sunset times
* 5-day weather forecast
* Multi-language support
* Modern glass-style UI
